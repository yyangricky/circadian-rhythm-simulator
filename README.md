# circadian-rhythm-simulator
Simulates the effects of sleep loss according to established circadian rhythm models.
This code uses `circadian` package which can be pip stalled using `pip install circadian`. Make sure to set this up before running the code. 
Our original code is implimented in actogram.py and amplitude_plot.py. To run these files, simply do `python actogram.py` and `python amplitude_plot.py`
